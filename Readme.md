# DevConf 2026 🚀

DevConf 2026 is a premium, modern single-page developer conference website built using semantic HTML5 and vanilla CSS3. The application features advanced CSS Grid systems, a responsive Bento layout for social proof, custom component styling for tiered pricing, and Font Awesome iconography tracking the industry's latest UI/UX trends.

🔗 **Live Demo:** [https://abdul-wahid789.github.io/DevConf-2026/](https://abdul-wahid789.github.io/DevConf-2026/)

---


### Typography & Icon Engine

* **Font System:** Google Fonts implementation tracking the variable weight axes of **Inter** (`100..900`).
* **Icons:** Inline integration via CDN Font Awesome Brand & Solid vector frameworks.

## 🚀 Getting Started Locally

To launch this project on your machine:

1. Clone the repository to your local directory:
```bash
git clone https://github.com/abdul-wahid789/DevConf-2026.git

```


2. Navigate into the root project directory:
```bash
cd DevConf-2026

```


3. Open `index.html` in your web browser of choice, or run it through an extension like VS Code's **Live Server** to observe real-time DOM changes.

## ⚡ Tech Stack

* **HTML5** (Semantic elements including `<header>`, `<main>`, `<article>`, `<nav>`, and `<section>`)
